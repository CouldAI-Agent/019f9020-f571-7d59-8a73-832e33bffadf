# PrepAI - AI Interview Simulator

An AI-driven mobile and web app that helps students and graduates prepare for real-world interviews. By inputting their specific course, domain, and optionally uploading their resume, users get tailored mock interview questions. They can practice answering via text or audio recording, and then receive a comprehensive dashboard report assessing their English level, technical knowledge, and overall market readiness.

## Features

*   **Custom Setup:** Users can enter their course (e.g., Computer Science, Marketing) and upload a resume to tailor the interview.
*   **Interview Modes:** Practice in a real-time mock interview setting or prepare with material-based pacing.
*   **Active Interview Screen:** A distraction-free environment to answer dynamically generated questions via text or voice recording.
*   **Comprehensive Reporting:** A post-interview dashboard showcasing Overall Score, English Level, and Market Readiness.
*   **Detailed Feedback:** Get qualitative feedback outlining strengths, areas for improvement, and a review of all answered questions.

## Tech Stack

*   **Framework:** Flutter (Mobile, Desktop, Web)
*   **Design System:** Material 3 with dynamic layouts for cross-platform responsiveness
*   **State Management:** Lightweight local state caching (easily extensible to BLoC or Provider)

## Getting Started

1.  Ensure you have Flutter installed.
2.  Clone the repository and run `flutter pub get`.
3.  Run `flutter run` to launch the app on your preferred platform (iOS/Android simulator, or Web).
